  Create project:
   composer create-project laravel/laravel:^8.0 LaravelEcommerceSingle
   cd LaravelEcommerceSingle/
   code .
  Package Setup Breeze
   composer require laravel/breeze --dev
   php artisan breeze:install
   php artisan migrate
   npm install
   npm run dev
